Welcome to DagsHub client's documentation!
============================================

This is the API reference for the `DagsHub Python Client <https://github.com/DagsHub/client>`_.

To install the client in your environment:

.. code-block:: console

    pip install dagshub

If you're working in a Google Colab/jupyter environment,
run ``pip install dagshub[jupyter]`` to install additional dependencies to enhance your experience.
